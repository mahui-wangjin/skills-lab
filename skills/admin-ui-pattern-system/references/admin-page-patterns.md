本文件用于在具体页面设计时加载。先根据业务任务选择主范式，再结合当前模板能力落地。

## 标准列表管理页

适用：用户、角色、任务、配置对象、策略、批次、运行记录等可搜索、分页、批量处理的对象。

默认结构：

- 可折叠适用备注/说明：只放范围、风险、数据边界、生产边界等必要提示；默认不挤占主工作区。
- Tabs：只在确实存在多个同层任务时使用，例如对象列表、版本、运行记录、审计；不要把不相关内容纵向堆叠。
- 搜索区：只放高频条件，低频条件折叠；按对象需要选择属性筛选、分类筛选、树筛选、状态分组、日期范围或快速搜索。搜索策略必须明确选择自动筛选或手动提交；自动筛选适合轻量查询，字段变化防抖提交，输入框回车提交，不再显示重复搜索按钮或 toolbar 搜索入口；手动提交适合昂贵查询或需要组合条件后统一提交的页面，并必须支持回车提交。
- 表格工具栏：新增、批量操作、隐藏/显示筛选、刷新、全屏、列设置、字段显示/隐藏、密度、固定列等能力按模板已有能力复用；不放和筛选区重复的搜索入口。
- 主表格：默认展示 P0 字段；操作列固定且收纳次要操作。
- 行内容：默认单行省略；阅读型列最多两行省略且行高自适应；完整文本通过快速查看浮层、详情抽屉或详情页承载。
- 行操作：最高频操作直接展示，其他进入更多菜单。
- 分页：跟随主表格，不把分页和其他内容隔开。
- 详情：默认使用抽屉；信息过多、需要多 Tab 或深链时使用详情页。
- 创建/编辑：默认使用抽屉；短确认或极轻量输入才使用弹窗；复杂流程使用步骤页或独立页。

通用控件要求：

- 标题帮助、工具栏图标、行操作、批量操作和更多菜单都必须有真实行为。
- 暂不可用功能必须显示为禁用态并给出原因；不能用但又不禁用的入口应移除。
- 不为“看起来专业”增加无内容帮助、空弹层、无响应图标或伪功能按钮。

默认线稿：

```text
[可折叠适用备注/说明]
[Tabs，如需要]
[筛选区：高频条件 + 折叠低频条件 + 属性筛选/快速搜索；明确自动筛选或手动提交]
[操作工具栏：新增/批量/刷新/全屏/字段设置/密度]
[列表区域：默认列 + 状态 + 行操作/更多]
[分页]
```

适合保留在主表字段：

- 名称/标题
- 状态
- 类型/分类
- 归属/范围
- 关键时间
- 风险等级或处理状态
- 少量高频操作

应进入详情或可选列：

- 长说明
- 原始 ID、trace、permission decision
- raw payload
- 审计历史
- 低频业务字段

项目内基线要求：

- 如果项目已有用户确认过的同类列表基线，后续列表必须复用其页面壳、搜索、工具栏、表格列、详情抽屉和长文本处理模式。
- 不得只做到“组件换成表格”就结束；必须让用户在截图上明显看到主任务、筛选、列表和详情承载变清晰。
- 常驻提示和指标卡只有在直接服务主任务时保留；否则折叠、移入详情或改为错误/边界状态时才出现。

## 树表主从页

适用：组织、部门、分类、本体分类、菜单、权限范围、数据域等层级结构。

默认结构：

- 左侧树/分类面板：搜索、展开/收起、节点操作。
- 右侧表格/详情：展示当前节点下对象。
- 节点操作和对象操作分离。
- 节点详情、对象详情、编辑表单默认通过抽屉承载；简单确认才用弹窗。

避免：

- 左侧树、右侧多个列表、下方表单同时堆叠。
- 节点配置和对象配置混在同一个表单里。

## 配置治理页

适用：规则、策略、字段映射、状态裁定、动作目录、权限策略、schema、本体版本。

默认结构：

- 配置对象列表：状态、版本、发布状态、风险等级。
- 配置详情、版本、diff、审计不要同时平铺；默认用 Tabs、抽屉或独立页分层承载。
- 详情/编辑入口：默认进入抽屉；字段多、影响范围复杂或需要 diff/审批上下文时进入独立详情页。
- 草稿编辑：独立编辑流，避免在列表里直接改高风险字段。
- diff/预览：发布前展示影响范围。
- 发布/回滚/禁用：高风险动作必须确认并审计。

必须表达：

- 当前线上版本
- 草稿状态
- 最近发布时间
- 修改人/审批人
- 影响范围
- 回滚路径

## 审批/流程处理页

适用：审批、复核、dry-run、批次执行、流程实例、异常处理。

默认结构：

- 队列/实例列表：状态、风险、创建时间、申请人/范围。
- 筛选区：状态、风险、时间、对象类型。
- 详情区域：证据、上下文、影响范围、历史操作。
- 操作区：批准、拒绝、重试、取消、提交执行等。
- 审计轨迹：抽屉、时间线或详情页内展示。

避免：

- 把所有动作计划都变成表格行内按钮。
- 把队列、证据、处理表单、审计轨迹和结果 payload 同时平铺。
- 执行动作缺少影响范围和后果说明。
- 只展示成功路径，不展示失败/部分失败。

## 图谱/画布工作台

适用：图谱编辑、本体编辑、流程编排、拓扑关系、节点边编辑。

默认结构：

- 画布主体：占据主要空间，默认不被说明、图例、添加表单常驻挤压。
- 顶部工具栏：保存草稿、撤销/重做、布局、适配视图、全屏、发布、回滚、diff。
- 左侧可选：对象列表、节点库、图层/分类。
- 右侧 inspector：只在选中节点/边后展示属性、关系、风险、审计信息；不是静态说明栏。
- 新增节点/关系：通过工具栏按钮触发，优先用抽屉或专门流程承载；短选择器可用弹窗。
- 图例/帮助：折叠、弹出或次级面板，不常驻挤压画布。

默认线稿：

```text
[标题/状态/保存/发布/回滚等主操作]
[可折叠说明/风险提示]
[Tabs：图谱编辑/节点列表/关系列表/版本 diff/审计，如需要]
[画布工具栏：新增节点/新增关系/布局/适配视图/全屏/图例/显示设置]
[大画布主体]
[选中节点或边后再打开 inspector/抽屉；新增流程用抽屉或专门流程]
```

必须避免：

- 画布很小，右侧表单和说明长期占据主要空间。
- 添加节点、添加关系、图例、说明、属性编辑全部同时显示。
- 手写简陋 SVG 作为生产级编辑器。

## 审计/观测页

适用：审计日志、权限决策、执行记录、任务轨迹、系统事件、错误记录。

默认结构：

- 时间、状态、用户、范围、对象类型筛选。
- 事件表格：紧凑、可扫描。
- 详情抽屉：完整 payload、上下文、关联对象；超长轨迹或多对象关联可升级为独立页。
- 轨迹视图：需要时用时间线或链路详情。
- 导出/复制：按权限开放。

注意：

- 原始 ID 和 payload 不应淹没默认列表。
- 失败原因、决策结果、风险等级要比底层字段更容易看到。

## 总览工作台

适用：运营总览、风险总览、任务总控、管理驾驶舱。

默认结构：

- 摘要指标：只放能驱动判断的少量指标，必须能下钻到明细。
- 待办队列：用户进入后最需要处理的事项。
- 风险/异常列表：按风险等级、时效、责任范围排序。
- 最近变化：新失败、新审批、新发布、新告警。
- 快捷入口：进入具体列表、处理台或详情页。

避免：

- 营销式 hero、大屏装饰、无意义卡片堆叠。
- 指标很多但没有处理路径。
- 把所有子模块压成一个长滚动首页。

## 分步向导

适用：策略创建、配置发布、导入确认、规则上线、批次发起。

默认结构：

- 步骤导航：基本信息、范围、规则/内容、预览、确认。
- 每步只处理一类决策。
- 步间校验明确，不能让错误堆到最后一步。
- 最终确认必须展示影响范围、风险、回滚方式和审计记录。

## 导入映射工作台

适用：Excel/CSV 导入、字段映射、数据预检、批量修复。

默认结构：

- 上传区。
- 字段映射区：源字段、目标字段、必填、转换、示例值。
- 校验结果：通过、警告、错误、不可导入。
- 错误明细：可筛选、定位、下载、重试。
- 提交/回滚：提交前展示影响范围和幂等/重复策略。

## 规则构建器

适用：规则、策略、权限条件、触发条件、通知条件。

默认结构：

- 条件组与逻辑关系。
- 可用字段/事实来源。
- 测试样例和命中解释。
- 影响范围预览。
- 高级 JSON/DSL 预览或只读模式。

避免：

- 只给 JSON 编辑器。
- 允许用户配置不存在或无权限的字段。
- 没有测试样例就允许发布。

## 运行监控页

适用：job、worker、调度、批次、异步执行、补偿任务。

默认结构：

- 运行实例列表：状态、进度、耗时、触发来源、负责人。
- 详情抽屉/页：输入、输出、日志、错误、重试、时间线。
- 操作：重试、取消、暂停、恢复、导出日志。
- 失败聚合：按错误类型、批次、对象范围聚合。

## Diff/发布确认页

适用：配置发布、本体版本、规则版本、权限策略、模板版本。

默认结构：

- 当前版本与目标版本。
- 新增/删除/修改分组。
- 影响范围、风险等级、审批记录。
- 发布、回滚、取消和保存草稿。

## 权限矩阵页

适用：角色权限、动作权限、字段权限、数据范围、菜单权限。

默认结构：

- 主体选择：用户、角色、组织或服务账号。
- 权限域：菜单、动作、字段、数据范围。
- 矩阵主体：支持分组、搜索、折叠和批量勾选。
- 预览：以某个用户查看最终权限。
- 审计：谁改了什么、何时生效、影响哪些对象。

避免：

- 一个超宽表横向滚动承载全部权限。
- 没有最终权限预览。

## 对象 360 详情页

适用：学生、任务、策略、批次、运行实例、配置对象等需要长期停留的详情。

默认结构：

- 摘要头：名称、状态、风险、关键操作。
- 关键事实：少量高频判断字段。
- Tabs：基础、关系、运行、审计、历史、权限、配置。
- 关联入口：跳转到相关列表或时间线。

## 异常复核台

适用：人工复核、异常补偿、状态确认、冲突处理。

默认结构：

- 待处理异常队列。
- 证据和上下文。
- 系统建议动作。
- 影响范围和风险提示。
- 处理表单和审计记录。

## 消息/通知运营页

适用：通知模板、发送批次、站内信、渠道触达、失败重试。

默认结构：

- 模板管理和发送记录分入口。
- 批次列表：渠道、范围、状态、成功/失败数。
- 接收人明细：触达状态、失败原因、重试记录。
- 内容预览和变量渲染。

## 搜索/资源选择器

适用：绑定资源、选择对象、批量关联、从大量数据中定位目标。

默认结构：

- 搜索筛选。
- 结果列表。
- 预览区。
- 已选区。
- 确认/取消。

大量对象选择不应塞进小弹窗；优先抽屉或独立选择页。

## 时间线/因果链路页

适用：状态解释、审计追踪、任务流转、跨系统事件。

默认结构：

- 时间线或链路图。
- 事件筛选。
- 事件详情。
- 关联对象和 trace。
- 操作者、来源系统、原因和结果。

## 日历/排期页

适用：排程、班次、考试安排、任务窗口、资源占用。

默认结构：

- 日历/时间轴主体。
- 资源或范围筛选。
- 冲突提示。
- 列表替代视图。
- 编辑/发布/回滚承载。

## 报表分析页

适用：运营报表、趋势分析、质量分析、风险分析。

默认结构：

- 口径说明和更新时间。
- 筛选区。
- 核心图表。
- 明细下钻。
- 导出权限和导出记录。

避免装饰型大屏；报表必须能解释指标口径和支持下钻。

## 模板/内容编辑页

适用：通知模板、文案模板、协议模板、渲染模板。

默认结构：

- 编辑器。
- 变量面板。
- 预览。
- 校验结果。
- 测试发送/测试渲染。
- 版本、发布和回滚记录。
